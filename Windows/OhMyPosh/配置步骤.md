# OhMyPosh 配置步骤

## 安装 OhMyPosh

```bash
winget install JanDeDobbeleer.OhMyPosh --source winget
```

## 配置 OhMyPosh

### 配置 OhMyPosh 个性化文件

1. 在 `.config` 目录下创建 `oh-my-posh` 目录
2. 将 [`config`](./config/) 下的文件复制到 `.config/oh-my-posh` 目录下

### 配置 OhMyPosh 终端自动执行

- 在`C:\Users\%USERNAME%\Documents\PowerShell\Microsoft.PowerShell_profile.ps1`中添加以下内容：

    ```powershell
    # Oh My Posh 配置（如果已安装）
    if (Get-Command oh-my-posh -ErrorAction SilentlyContinue) {
        oh-my-posh init pwsh  --eval --config "C:/Users/ender/.config/oh-my-posh/config.json" | Invoke-Expression
    }
    ```
