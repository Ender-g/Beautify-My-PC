# fastfetch 配置步骤

## 安装 Fastfetch

```bash
winget install Fastfetch
```

## 配置

### 配置 fastfetch 个性化文件

1. 在 `.config` 目录下创建 `fastfetch` 目录
2. 在 `fastfetch` 目录下运行 `git clone https://github.com/fastfetch-cli/fastfetch.git` 克隆仓库
3. 在 `fastfetch` 目录下创建 [`ascii.txt`](./config/ascii.txt) 和 [`config.jsonc`](./config/config.jsonc) 文件
4. 复制[`config.jsonc`](./config/config.jsonc) 文件内容到 `config.jsonc` 文件中，并更改配置中的 `"%USERNAME%"` 为你的用户名
5. 创建 `ascii.txt` 文件并复制[`ascii.txt`](./config/ascii.txt) 文件内容到 `ascii.txt` 文件中

### 配置 fastfetch 终端自动执行

#### 配置 powershell 自动执行

- 在`C:\Users\%USERNAME%\Documents\PowerShell\Microsoft.PowerShell_profile.ps1`中添加以下内容：

    ```powershell
    # 强制每次使用你自己的 Fastfetch 配置（绕过路径混乱问题）
    if (Get-Command fastfetch -ErrorAction SilentlyContinue) {
        echo "Welcome back to your terminal!!! :)"
        fastfetch -c "C:/Users/%USERNAME%/.config/fastfetch/config.jsonc"
        echo "Have a nice day!"
    }
    ```

#### 配置 cmd 自动执行

1. 使用快捷键 `Win+R` 打开运行窗口，执行 `regedit` 命令打开注册表
2. 在 `计算机\HKEY_CURRENT_USER\Software\Microsoft\Command Processor`（如果不存在 `Command Processor` 项，直接新建该项即可） 中添加 `AutoRun` 字符串值，并设置值为 `cls && echo Welcome back to your terminal!!! :) && fastfetch`
